# [Live Preview](https://post-app-mf.netlify.app/)
	
## Technologies
Client
* React
* Redux
* Axios
* Redux-thunk 
* Material-UI

Server
* Node.JS
* Express
* MongoDB

## Setup
To run this project, install it locally using npm:
```
cd server
```
```
npm install
```
```
npm start
```
## and
```
cd client
```
```
npm install
```
```
npm start
```
